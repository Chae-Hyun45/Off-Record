# Off Record

> AI 기반 하루 감정 기록 및 관리 앱  
> 사용자의 감정, 스트레스, 피로도, 수면 상태, 일기 내용을 기록하고 날짜별로 확인할 수 있는 Android 프로젝트입니다.

<br/>

## 📌 프로젝트 소개

**Off Record**는 사용자가 하루 동안 느낀 감정과 상태를 간단하게 기록하고, 캘린더와 기록 리스트를 통해 다시 확인할 수 있도록 만든 감정 기록 앱입니다.

기존의 단순 일기 앱과 달리, 감정 점수와 객관식 질문을 함께 입력하여 사용자의 하루 상태를 더 구체적으로 남길 수 있도록 구성했습니다.  
추후에는 기록된 데이터를 기반으로 AI 분석 및 피드백을 제공하는 감정 관리 서비스로 확장하는 것을 목표로 합니다.

<br/>

## 🎯 프로젝트 목표

- 사용자가 부담 없이 하루 감정을 기록할 수 있는 입력 흐름 제공
- 감정, 스트레스, 피로도, 수면 등 다양한 상태 정보 저장
- 캘린더 기반으로 날짜별 기록 확인
- 월별 기록 리스트 및 상세 조회 기능 제공
- 감정 기록 데이터를 바탕으로 통계 및 분석 화면 제공 예정
- 알림, 데이터 관리 등 사용자 설정 기능 확장 예정
- 향후 AI 분석 및 감정 리포트 기능으로 확장 가능한 구조 설계

<br/>

## 📱 주요 기능

### 1. 캘린더 화면

캘린더 화면에서는 날짜별 감정 기록 여부를 확인할 수 있습니다.

사용자가 특정 날짜를 선택하면 해당 날짜에 저장된 감정 기록 요약이 표시되며, 기록이 없는 날짜는 `?`와 함께 기록 없음 상태로 표시됩니다.

- 오늘 날짜 기준 캘린더 표시
- 날짜 선택 시 해당 날짜 기록 조회
- 기록이 있는 날짜는 감정 이미지로 표시
- 기록이 없는 날짜는 `?`로 표시
- 선택 날짜의 감정, 점수, 스트레스, 일기 요약 표시

<br/>

### 2. 통계 화면

통계 화면은 사용자가 기록한 감정 데이터를 기반으로 감정 변화와 상태 흐름을 확인하기 위한 화면입니다.

현재는 통계 탭 화면 구조를 구성한 상태이며, 추후 저장된 기록 데이터를 활용하여 감정 점수 평균, 기록 횟수, 스트레스 빈도 등을 제공할 예정입니다.

- 월별 감정 기록 통계 제공 예정
- 평균 감정 점수 표시 예정
- 스트레스 및 피로도 빈도 분석 예정
- 감정 변화 흐름 시각화 예정

<br/>

### 3. 감정 기록 입력 화면

감정 기록 입력 화면에서는 사용자가 오늘의 감정과 하루 상태를 기록할 수 있습니다.

사용자는 먼저 대표 감정 이모지를 선택하고, 이후 세부 질문에 답변하여 감정 기록을 작성합니다.

입력 항목은 다음과 같습니다.

- 오늘의 대표 감정
- 감정 점수
- 감정에 영향을 준 요인
- 스트레스 정도
- 피로도
- 수면 상태
- 오늘 필요한 것
- 받고 싶은 피드백 유형
- 식사 여부
- 일기 내용

<br/>

### 4. 내 기록 리스트 화면

내 기록 리스트 화면에서는 저장된 감정 기록을 월별로 구분하여 확인할 수 있습니다.

각 기록은 카드 형태로 표시되며, 기록 카드를 클릭하면 해당 날짜의 상세 기록 화면으로 이동합니다.

- 저장된 기록을 월별로 구분하여 표시
- 감정 이미지, 날짜, 일기 내용, 점수 표시
- 기록 카드 클릭 시 상세보기 화면으로 이동
- 이전 기록을 한눈에 확인 가능

<br/>

### 5. 기록 상세보기 화면

기록 상세보기 화면에서는 선택한 날짜의 감정 기록을 자세히 확인할 수 있습니다.

- 날짜 및 기록 시간 표시
- 감정 이미지와 감정명 표시
- 감정 점수 표시
- 일기 내용 표시
- 스트레스, 피로도, 수면, 필요한 것, 피드백 등 세부 입력값 표시

<br/>

### 6. 설정 화면

설정 화면은 앱 사용 환경과 데이터 관리 기능을 제공하기 위한 화면입니다.

현재는 설정 탭 화면 구조를 구성한 상태이며, 추후 알림 설정, 데이터 초기화, 앱 정보 확인 등의 기능을 추가할 예정입니다.

- 기록 알림 설정 기능 추가 예정
- 저장된 기록 초기화 기능 추가 예정
- 앱 정보 및 버전 정보 표시 예정
- 사용자 설정 관리 기능 확장 예정

<br/>

## 🧭 앱 화면 흐름

```text
IntroActivity
    ↓
MainActivity
    ├─ CalendarFragment        캘린더 화면, 날짜별 기록 조회
    ├─ StatsFragment           통계 화면, 감정 분석 기능 확장 예정
    ├─ 기록 버튼 클릭
    │     └─ Emotion Select Dialog
    │           └─ InputFragment  감정 기록 입력 화면
    ├─ ExtraFragment           내 기록 리스트 화면
    │     └─ RecordDetailFragment  기록 상세보기 화면
    └─ SettingsFragment        설정 화면, 알림 및 데이터 관리 기능 확장 예정
```
